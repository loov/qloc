`qloc` does a quick line count over files.

Example output for this folder:

```
extension           files       binary        blank         code
----------------------------------------------------------------
go                      3            0           43          211
                        1            0            5           20
gitignore               1            0            5           19
md                      1            0            1            5
----------------------------------------------------------------
summary                 6            0           54          255
```

To install:
```
go get github.com/loov/qloc
```
